# NISCH-RESTO | Premium Fine Dining Website

A modern, luxurious, and fully responsive restaurant website designed for NISCH-RESTO, a premium fine-dining establishment. This project showcases elevated culinary artistry through a sleek user interface, smooth animations, and a seamless reservation experience.



## 🍽️ Key Features

- **Luxury Design Aesthetic**: A premium look and feel using curated typography (Playfair Display & Montserrat) and a sophisticated color palette.
- **Responsive Layout**: Fully optimized for desktops, tablets, and smartphones.
- **Interactive Menu**: Highlights signature dishes like Coastal Scallops, Black Truffle Linguine, and Aged Wagyu Filet.
- **Reservation System**: A functional booking form for guests to reserve their experience.
- **Team Showcase**: Introduces the culinary masters behind the restaurant.
- **Testimonials & Gallery**: Builds trust and showcases the atmosphere of the dining room.
- **Dynamic Navigation**: Smooth scroll behavior and a mobile-responsive toggle menu.

## 🛠️ Built With

- **HTML5**: For semantic structure.
- **CSS3 (Vanilla)**: Custom styling with CSS variables, Flexbox, and Grid.
- **JavaScript**: For interactive components, scroll effects, and form handling.
- **Google Fonts**: [Montserrat](https://fonts.google.com/specimen/Montserrat) & [Playfair Display](https://fonts.google.com/specimen/Playfair+Display).
- **Font Awesome**: High-quality icons for social media and UI elements.

## 🚀 Getting Started

To run this project locally:

1. Clone or download the repository.
2. Ensure you have the following assets in the same directory:
   - `index.html`
   - `style.css`
   - `script.js`
   - Image assets: `hero.png`, `scallops.png`, `pasta.png`, `wagyu.png`.
3. Open `index.html` in any modern web browser.
